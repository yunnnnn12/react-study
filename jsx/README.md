# JSX

## What I Learned
## 1. jsx.js
- React 컴포넌트는 JSX 문법을 사용하여 마크업을 나타내고, 서로 마크업과 렌더링 로직을 함께 그룹화한다.
- 한 컴포넌트에서 여러 엘리먼트를 반환하려면, 하나의 부모 태그로 감싼다. JSX는 JS객체로 변환되기 때문에 하나의 배열로 감싸지 않은 하나의 함수에서는 두개의 객체를 반환할 수 없다. 
- 모든 태그는 닫아주어야 한다.
- 변수명은 카멜 문법을 적용한다.

### HTML(JSX) 문법
 - h1 : 제목
 - div : 의미를 가지지 않는 태그로, 어떤 내용이든 묶어내는 기능 
 - src(source) : source라는 속성을 사용해서 삽입할 때 이미지의 경로를 지정
 - alt(alternative) : 이미지를 출력하지 못하는 상황에 이미지 대신 보여질 텍스트를 지정
 - class(className) : 별명
 - ul : unordered list. 순서 없는 리스트
 - li : 리스트 항목  

 ## 2. jsx_2.js
 - src또는 alt를 동적으로 지정하려면 ""를 {}로 바꾸어야 한다. 
 - jsx에서 객체를 전달하려면 person={{ name: "Hedy Lamarr", inventions: 5 }}와 같이 다른 중괄호 쌍으로 객체를 감싸야 한다. 
 - 리액트에서는 style="color: red; font-size: 20px"> 이러한 인라인 스타일이 필요없다.
 - 인라인 스타일 프로퍼티는 캐멀 케이스로 작성된다.
