# List Rendering

## What I Learned
### 1. map.js
- js에서 배열은 "[]"로 감싸져 있고 값들이 ","로 구분되어 있다. 
- map은 배열을 반환하는 함수이다. 
- `person => <li>{person}</li>`는 함수이다. 
- listItems는 people배열의 요소를 하나 꺼내서 li태그로 감싸서 배열로 만드는 것이다. 

### 2.filter.js
- 화살표 함수는 암시적으로 => 바로 뒤에 식을 반환하기 때문에 return 문이 필요하지 않다. 하지만 => 뒤에 중괄호가 오면 return을 작성해야 한다. 
- "b"는 js에서 그대로 쓰는 html태그이다. 굵게(bold)표시하는 html태그이다.
